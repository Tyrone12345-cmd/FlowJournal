# Welcome Sound

Platziere hier eine `welcome.mp3` Datei für den Login-Sound-Effekt.

Empfohlene Sound-Quellen:
- https://freesound.org/ (kostenlose Sounds)
- https://mixkit.co/free-sound-effects/ (Gaming Sounds)
- Suche nach: "game intro", "level up", "success fanfare", "achievement unlock"

Empfohlene Einstellungen:
- Format: MP3
- Dauer: 1-2 Sekunden
- Lautstärke: mittellaut (wird im Code auf 30% reduziert)
